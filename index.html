👥 義工/學員名單庫

在下方框中貼入您的義工或學長名單（用逗號或換行隔開），下方各執事分配下拉選單會自動同步，方便快速點選！

💡 如何保存與更新網頁？

  1. 在右側各個典禮流程中，點擊下拉選單或手動輸入人名。
  2. 排定完畢後，點擊上方「匯出並下載 data.js」按鈕。
  3. 將下載的 data.js 覆蓋掉本專案資料夾中的舊 data.js。
  4. 使用 Git 提交變更並推送至您的 GitHub，Vercel 將自動重新發布！
